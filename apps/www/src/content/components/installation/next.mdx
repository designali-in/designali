<Callout>

**If you're using Next.js 15, see the [Next.js 15 + React 19](/docs/react-19)
guide.**

</Callout>

<Steps>

<Step>Create project</Step>

Run the "init" command to create a new Next.js project or to setup an existing
one:

```bash
npx shadcn@latest init
```

<Callout className="mt-4">

You can use the -d flag for defaults i.e new-york, zinc and yes for the css
variables.

```bash
npx shadcn@latest init -d
```

</Callout>

<Step>Configure components.json</Step>

You will be asked a few questions to configure components.json`:

```txt showLineNumbers
Which style would you like to use? › New York
Which color would you like to use as base color? › Zinc
Do you want to use CSS variables for colors? › no / yes
```

<Step>That's it</Step>
</Steps>
You can now start adding components to your project.

```bash
npx shadcn@latest add button
```
